✨ v38 업데이트 안내

• 데이터 자동 정렬: 등록 현황 리스트가 동 -> 호수 -> 회원번호순으로 자동 SORT됩니다.

• 다크모드 지원: 상단 다크버튼을 통해 눈피로 방지 다크모드로 전환할 수 있습니다.

• 유튜브 링크: 프로그램 제작에 도움이 됩니다. 서명클릭후 유튜브 채널(구독,좋아요)

• 카카오 문의 추가: 우측 상단에 카카오채널 [문의] 버튼이 추가되었습니다.

• 세대별합산요약: 등록인원전체표시,회원번호.전화번호표시,세대전체삭제버튼추가.엑셀내보내기수정.

• 데이터삭제주의: PC를 교체하거나 브라우저 초기화시 데이터 내보내기(JSON)', '데이터 가져오기', '엑셀 파일 다운로드' 기능을 꼭 사용하십시요
