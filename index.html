✨ v44 업데이트 안내

• 버그 수정 (v44): 아래 항목들이 수정되었습니다.

• 동 드롭다운 동기화: 환경설정에서 동 수 변경 시 모든 동 선택 드롭다운이 즉시 갱신됩니다.

• 전체 세대 자동생성: 648세대 고정이 아닌 환경설정의 동 수 / 세대 수를 기반으로 동적 생성됩니다.

• 호수 정렬 오류 수정: 조회/고지서 출력에서 호수가 문자순(201 < 99) 대신 숫자순(99 < 201)으로 올바르게 정렬됩니다.

• 항목 선택 필터 적용: 부과 통합 조회에서 '헬스장/주차/사물함' 항목 선택이 실제 조회 결과에 반영됩니다.

• 해지 관리 페이지 스타일 수정: 헬스장/주차/사물함 해지 관리 페이지 헤더가 정상 표시됩니다.

• 세대 DB 드롭다운 누락 수정: 초기 로드 시 세대 DB 필터 드롭다운도 올바르게 초기화됩니다.

• 데이터삭제주의: PC를 교체하거나 브라우저 초기화시 '데이터 내보내기(JSON)', '데이터 가져오기', '엑셀 파일 다운로드' 기능을 꼭 사용하십시오.
